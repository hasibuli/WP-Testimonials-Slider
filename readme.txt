== Description ==
WordPress Portfolio Plugin is a wordpress portfolio plugin. Use this shortcode [WPTS-TESTIMONIAL] in the post/page" where you want to display slider! 

The plugin is designed for portfolio site. "WP Testimonials Slider" is a free wordpress plugin.


= Usage =

* Go to your Dashboard after installation and navigate to "Testimonial >> Testimonial Settings" to configure the portfolio.

= Features =

  * Very easy installation
  * Shortcode system
  * Unlimited color variation by using jQuery color picker.
  * Responsive design
  * Custom post supported
  * Flexible and easy to use
  * Lightweight JQuery
  * Plugin tested with IE6+, FF 20+, Chrome, Safari
  * Fully Custom post type & custom settings.


== Installation ==
1. Upload `wp-testimonial-slider` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the new menu item "Testimonial" and create your testimonials.
3. Just copy and paste " <strong>[WPTS-TESTIMONIAL]</strong> in the post/page" where you want to display testimonials.
